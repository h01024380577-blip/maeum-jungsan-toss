하나만 선택

가장 중요한 이유 하나만 남깁니다. 데이터 분석은 단순하지만 실제 이유가 여러 개인 경우 표현이 부족할 수 있습니다.

B

여러 개 선택 가능

복합적인 이유를 담을 수 있습니다. UI에는 선택 개수와 해제 상태가 명확히 보여야 합니다.

C

칩 하나 + 직접 입력 보완

대표 이유 하나를 고르고, 필요한 경우 직접 입력으로 보완합니다. 분석과 자유도를 절충합니다.
