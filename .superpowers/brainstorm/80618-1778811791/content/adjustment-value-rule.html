금액 차액을 기억하는 방식은 제외합니다. 저장할 prior는 사용자가 선택한 이유와 그 이유에서 파생되는 상대방 context입니다.

문제
한 번의 선택 금액을 다음 추천에 기계적으로 복사하게 됩니다.

이 방식은 사용자의 이유를 학습하지 못하므로 제외합니다.

저장할 것
사용자가 고른 이유
동행 인원까지 고려해서
상대방 context
이 사람에게는 다음 추천 때 동행 인원 prior를 함께 고려

다음 추천은 이 이유 context를 입력으로 받아 다시 계산합니다.

다음 추천에서의 사용 방식
기존 추천 prior + 상대방 이유 context
관계·과거 이력·장소로 기본 추천을 만들고, 같은 상대방에게 저장된 이유 context를 추가 입력으로 넣어 추천을 다시 계산합니다.
