AI 추천 금액을 사용자가 수정하여 저장한 경우, 변경 이유를 묻는 UX를 함께 설계해요.

현재 흐름 (HomeTab.tsx)
현재 저장 시 동작

1. 청첩장 URL/사진 분석 → AI가 amt 추천 + recommendationReason 부여

2. 사용자가 바텀시트에서 AmountReviewCard로 금액 검토/수정

3. "저장하기" → handleSave → addEntry (DB 저장)

4. 토스트 "저장 완료!" + (계좌 있으면) 송금 모달

※ 현재 initialParsedData로 변경 여부는 추적 중이지만, 사유는 묻지 않음
